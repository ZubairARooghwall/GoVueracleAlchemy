# Simple File Management System

This repository is for Database Administration course in 4th semester. 

## Technologies

### Backend

* Go langauge
* Gin backend framework
* WebRTC
* V9 for compression
* FFMPEG for working with multimedia
  
### Database

* Oracle Database Express Edition
  
### Frontend

* HTML
* CSS
* Javascript
* Vue.js
